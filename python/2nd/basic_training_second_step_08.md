# 8　range() 関数の引数のセット方法のバリエーション

ファイル名は2_8_month.py
```python=
# 1月から12月までループ文で表示させるプログラム
for i in range(1, 13):
    print(i, '月')
```

## コードの解説
|No|行|対象|説明|
|---|---|------------|---|
| 1|2|`for i in range(1, 13):`|range(1, 13) と書いてあるので、取り出す数値は、1, 2, 3, 4, 5, 6, 7, 8, 7, 9, 10, 11, 12 となります。<br/>range(1,13)は、1以上13未満の数を取り出せ、という意味です。|
